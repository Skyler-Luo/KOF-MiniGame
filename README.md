# 🎮 KOF-MiniGame

一个拳皇双人对战小游戏 👊

## ✨ 功能特性

- 🕹️ 双人本地对战
- 🎨 GIF 动画角色渲染
- ⏱️ 60秒倒计时
- ❤️ 血条系统
- 📱 全屏自适应

## 🎯 操作说明

| 操作 | 玩家1 | 玩家2 |
|------|-------------|-------------|
| 跳跃 | W | ↑ |
| 左移 | A | ← |
| 右移 | D | → |
| 攻击 | 空格 | Enter |

## 🚀 快速开始

1. 克隆项目
```bash
git clone https://github.com/Skyler-Luo/KOF-MiniGame.git
```

2. 使用本地服务器运行（如 Live Server）

3. 打开浏览器访问 `index.html`

## 📁 项目结构

```
KOF-MiniGame/
├── static/
│   ├── css/          # 样式文件
│   ├── js/           # JavaScript 模块
│   │   ├── ac_game_object/  # 游戏对象基类
│   │   ├── controller/      # 键盘控制器
│   │   ├── game_map/        # 游戏地图
│   │   ├── player/          # 玩家角色
│   │   └── utils/           # 工具类
│   └── images/       # 图片资源
└── templates/        # HTML 模板
```
